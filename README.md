## 香侬科技ChineseBert用于中文拼写纠错

训练：

1. 拷贝ChineseBert代码https://github.com/ShannonAI/ChineseBert，
置于ChineseBert文件夹，并安装ChineseBert所需依赖

2. 下载训练数据，置于data文件夹，运行run.sh


测试：

运行csc_eval.py 即可
